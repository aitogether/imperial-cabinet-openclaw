# USER.md - 用户画像

> **使用说明**: 复制为 `USER.md` 放入你的 OpenClaw workspace，填入你自己的信息。

## 服务对象

- **身份**: （填入，例如：个人开发者、独立创业者、小团队负责人）
- **技术水平**: 熟悉命令行和基本开发，但不想亲自写每一个命令
- **工作方式**: 偏好「批示」与「拍板」，将执行交给 AI Agent

## 终极目标

用 AI 打造高效的工作流 / 一人公司，最大化减少重复性劳动，建立稳定、可扩展的自动化产出。

## 性格与偏好

- **实用主义**: 讨厌空谈和花架子
- **方案偏好**: 简单可靠、易维护，「够用就好」，拒绝为微小收益引入巨大复杂度
- **决策标准（三重审视）**:
  1. 技术可行性
  2. 商业可行性 / 变现路径
  3. 风险与维护成本（尤其是对时间精力的占用）

## 沟通偏好

- **语言**: 默认简体中文
- **输出风格**: 少废话多产出，每轮互动尽量产出实物成果（脚本、文档、计划、报告等）
- **格式**: 输出结构化，标题、列表、表格优先
- **回复长度**: 默认短答，信息密度优先
