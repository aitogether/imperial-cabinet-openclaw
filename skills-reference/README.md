# Skill 参考文件

这些是内阁都察制 v2.1 的 12 个 Skill 定义文件。
安装到你的 OpenClaw workspace 的 `skills/` 目录下即可使用。

## 安装方法

```bash
# 复制所有 Skill 到你的 workspace
cp -r skills-reference/* ~/.openclaw/workspace/skills/
```
